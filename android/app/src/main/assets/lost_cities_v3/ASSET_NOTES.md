# Asset notes

## Visual changes in this pass

- Suit bands are more saturated for faster scan speed.
- The former gray route is now pure white, with icy accents only as support.
- Large center numerals use a black fill plus white and black double outline.
- Bottom expedition-name text has been removed from card faces.
- Dark-stone app backgrounds are included for dark-mode UI work.

## Files

- `assets/cards/png/` contains all card face and card back PNG files.
- `assets/backgrounds/` contains high-resolution dark-stone backgrounds.
- `assets/previews/board_preview.png` shows a compact full-board layout.
- `assets/previews/contact_sheet_base_60.png` shows the base deck.
- `assets/previews/contact_sheet_full_72.png` shows the full deck including purple.

## Readability choices

The cards are optimized for small on-screen presentation:

- the visible top band carries the suit through color and icon
- corner values are isolated in dark capsules
- center numerals are intentionally oversized
- artwork is pushed away from the lowest-clutter reading zone
- the bottom edge is text-free so stacked cards stay clean

## Suggested UI usage

- Use the top ~18 percent of each card as the "must remain visible" overlap area in expedition columns.
- Keep discard piles slightly fanned or offset so players can still identify the top card quickly.
- Use the card back for the draw pile and hidden opponent hand.
- The included stone backgrounds are suitable for dark-mode boards and menus.

## Wager card convention

All three wager cards within a suit intentionally use the same visual treatment because they are functionally identical.
